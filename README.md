# yKool

## Tarkvaraarenduse praktika projekt

### Moodulite alla laadimine ja koodi käivitumine

#### API käivitumine

---

```
cd api
npm install
npm start
```

#### Kliendi poole käivitumine

---

```
cd client
npm start
```
